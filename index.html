Welcome to the Mood Board



Here you will find an intersection between how you're feeling at the moment and a music playlist to match that feeling. Below you will see a textbox. We invite you to express yourself in a sentence and submit away. A link will then be generated. This link will take you to a spotify playlist that matches how you're feeling.

A Look into the Future:

Gauge User's Desired Mood
In the next version of our application, we would like to first ask the user how they are feeling. Then, we would like to ask the user if they want a playlist that matches this emotion, or if they would rather have a playlist recommended based on how the user would like to feel.

Playlist to Generate on our Application
In the next version of our application, we would like to keep the user on our web application and have them listen to the playlist directly on our site instead of being diverted to Spotify. This will hopefully make the user experience more personal.

Add Color Themes to Playlist Page
Once we are able to display playlists on our website, we would like the user to be able to toggle between different color themes researched specifically to fit either their current or desired emotion.

Add More Media for User to Choose From
Once we perfect the user experience with music as the main media source, we would then like to expand our media sources into literature, movies, guided meditations and so on.
